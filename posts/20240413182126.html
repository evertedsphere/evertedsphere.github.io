I currently use Zotero 7.

1. Vanilla Zotero configuration

1.1. Linked files

Set Zotero’s linked files dir to either your org-roam directory or a subdir: for me this is ~/o/kb/refs, where ~/o/kb is my org-roam directory.

1.2. File renaming

Set up file renaming according to your preferences; I dislike spaces in filenames due to the minor inconveniences they pose when using shells. I use 

{{ firstCreator case="snake" suffix="__" }}{{ year suffix="__"}}{{ title truncate="80" case="snake" }}

2. Sync your Zotero library to a BibTeX file

I’m using test build 6.7.173.2829.6002 of Better BibTeX. Set up an automatic export to ~/o/kb/refs/refs.bib.

3. Export Zotero files into the Roam folder

The solution until Zotero 6 used to be Zotfile, which is no longer maintained. Attanger, which is referenced in a lot of forum posts, does not work for me and it appears some others: see MuiseDestiny/zotero-attanger#83. I used Zotmoov (version 1.0.0.0), setting it up to export to ~/o/kb/refs.
